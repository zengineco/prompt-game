# 🗺️ PROMPT — Roadmap

> The vision: the biggest metaverse party game on the internet.
> This is where we're going.

---

## ✅ v1.0 — Discord MVP (Current)

- [x] Core game loop: Prompt → Respond → Vote → Score → Rotate
- [x] 150 built-in PROMPT cards
- [x] Anonymous submission & voting
- [x] Dealer rotation (poker-style)
- [x] Tie-breaking by Dealer
- [x] 5-minute submit / 1-minute vote timers
- [x] Auto-advance on timeout
- [x] Per-server isolated games
- [x] 3–9 players
- [x] Slash commands

---

## 🔜 v1.1 — Quality of Life

- [ ] `/prompt-config` command — server admins set custom round timers, target score, max players
- [ ] Spectator mode — watch without playing, no submissions
- [ ] "Skip prompt" mechanic — Dealer can burn a prompt and draw a new one (once per round, costs nothing)
- [ ] Emoji reactions for voting instead of buttons (accessibility)
- [ ] Auto-cleanup — delete game messages after session ends to keep channel clean
- [ ] `/prompt-help` — in-Discord full rules explanation
- [ ] Persistent leaderboard per server (top scorers all-time)

---

## 🔜 v1.2 — Deck Expansion System

- [ ] **Theme packs** — curated sub-decks: 🍷 *Wine Mom Edition*, 💼 *Corporate Confessions*, 🧒 *Parenting Fails*, 💔 *Relationship Red Flags*
- [ ] **Community prompt submissions** — players submit prompts during game with `/prompt-suggest`, mod approves
- [ ] **Custom deck builder** — server admins upload their own PROMPT card lists via a text file
- [ ] **NSFW deck** — opt-in adult prompts (Discord NSFW channel gating)
- [ ] Prompt rating system — after each round, players rate the prompt 1–5 for quality; low-rated prompts get retired

---

## 🔜 v1.3 — The Web Table (Big One)

A standalone **web app** companion to the Discord bot:

- [ ] **Poker-table interface** — 2D top-down table view with seats, player avatars, dealer chip animation
- [ ] **Seat selection** — "change seats" feature, drag yourself around the table
- [ ] **Live game feed** — spectators watch submissions come in (status only, not content) with a ticker
- [ ] **Mobile-first response input** — tap to open text box, minimal and unobtrusive
- [ ] **Anonymous reveal animation** — cards slide to center, flip to reveal on vote close
- [ ] **Scoreboard sidebar** — always visible, animated point increments
- [ ] **Discord OAuth** — log in with Discord, carries your score and identity
- [ ] Can be played entirely in web without Discord (standalone mode)

---

## 🔜 v2.0 — Live Rooms & Spectators

- [ ] **Room codes** — generate a 6-character code, friends join from any platform
- [ ] **Twitch/YouTube integration** — streamers can expose voting to chat (read-only, chat votes count as one combined vote)
- [ ] **Audience mode** — unlimited spectators watch live with real-time reaction emojis
- [ ] **Stream overlay** — OBS browser source showing current prompt + vote status in stylized overlay
- [ ] **"Hot seat" mode** — one player answers ALL responses anonymously; everyone votes on which they think is the real human vs. which is AI-generated

---

## 🔜 v2.1 — AI Integration (The Meta Layer)

The game becomes about separating humans from AI. Players don't just respond — they try to *write like AI* to fool voters, and voters try to spot the real AI.

- [ ] **One response per round is generated by actual Claude/GPT** — players vote not knowing which is real AI
- [ ] **"Spot the Bot" bonus** — if you correctly identify the AI's response, +1 point
- [ ] **AI difficulty settings** — easy (obvious AI), hard (indistinguishable), nightmare (AI has read your past responses and mimics you)
- [ ] **Hallucination Mode** — AI responses are intentionally wrong/absurd; players try to out-hallucinate the bot
- [ ] **Post-round reveal** — "This one was actually AI. Here's why it was or wasn't convincing."

---

## 🔜 v2.2 — Social & Viral Features

- [ ] **Shareable round recaps** — generate a stylized image of the funniest response from each round, post to Twitter/Instagram
- [ ] **Response Hall of Fame** — per-server museum of the most-voted responses ever
- [ ] **Cross-server tournaments** — bracket-style competitions between Discord servers
- [ ] **Daily PROMPT** — one prompt published every day on social media; anyone can reply on web; top response featured tomorrow
- [ ] **Prompt of the Week** — community votes on the best new submitted prompt; winner enters official deck

## 🔜 v3.0 — The Platform

- [ ] **Monetization** — server subscriptions unlock premium theme packs, custom branding, larger player counts
- [ ] **Creator marketplace** — community creators publish and sell theme packs
- [ ] **PROMPT API** — other bots and apps can pull from the prompt database
- [ ] **Mobile apps** — native iOS/Android for the web table experience
- [ ] **Accessibility** — screen reader support, colorblind modes, keyboard-only navigation
- [ ] **Localization** — prompt decks translated and culturally adapted: Spanish, French, Portuguese, Japanese editions
- [ ] **Corporate/party license** — white-label for team-building events, bachelorette parties, conference ice-breakers

---

## 🌌 Long-Term Vision

PROMPT is ultimately a game about the weirdly intimate relationship humans have with AI —
the questions we'd never ask another person, the confessions we make to a machine at 2am,
the embarrassing gaps in knowledge we're too proud to admit to friends.

The endgame isn't just a party game. It's a living archive of human curiosity, insecurity,
and humor — crowdsourced from the millions of real queries typed into chatbots every day.

**The biggest metaverse converter of all time** starts with one embarrassing question
and the courage to submit it anonymously.
